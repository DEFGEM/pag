# PAG — Aplicación frontend (React + TypeScript + Vite)

Resumen
-------
Repositorio que contiene una aplicación web frontend basada en React + TypeScript y Vite. Usa Tailwind CSS con el tema shadcn y múltiples componentes UI.

Stack
-----
- Node.js 20
- React 19
- TypeScript 5.9
- Vite 7
- Tailwind CSS

Rápido inicio
-------------
1. Ir al directorio de la app:
   cd app
2. Instalar dependencias (npm):
   npm install
3. Levantar servidor de desarrollo:
   npm run dev
4. Construir producción:
   npm run build
5. Lint:
   npm run lint

Estructura principal
--------------------
- app/src/        — código fuente (components, hooks, sections)
- app/public/     — assets públicos (imágenes, favicon)
- app/package.json
- app/README.md   — info específica de la plantilla

Notas
-----
- El proyecto incluye package-lock.json → se gestiona con npm.
- Revisar compatibilidad de dependencias si se actualiza React o TypeScript.
