# App — Frontend (React + TypeScript + Vite)

Descripción
-----------
Carpeta que contiene la aplicación React. Basada en la plantilla Vite + TypeScript y configurada con Tailwind CSS (tema shadcn).

Cómo desarrollar
----------------
1. Instalar dependencias desde la raíz o dentro de app:
   cd app
   npm install
2. Iniciar desarrollo:
   npm run dev
3. Build:
   npm run build
4. Lint:
   npm run lint

Estructura
----------
- src/ — código fuente
  - components/ — componentes UI
  - hooks/ — hooks personalizados
  - sections/ — secciones de página
  - main.tsx, App.tsx — punto de entrada
- public/ — assets públicos
- vite.config.ts — configuración de Vite
- tailwind.config.js — configuración de Tailwind

Notas
-----
- Recomendado usar Node 20.
- Si trabajas con VSCode, instala extensiones de TypeScript y ESLint.
